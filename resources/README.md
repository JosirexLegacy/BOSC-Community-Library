\#  Resources



This folder contains curated tools, links, and references useful for open-source contributors.



\---



\##  Suggested Categories



\- \*\*Git \& GitHub\*\* — workflows, cheat sheets, best practices

\- \*\*Documentation\*\* — tools for writing good docs

\- \*\*Community\*\* — OSS communities to join and learn from

\- \*\*Beginner Guides\*\* — where to start in open source



\---



\##  Adding a Resource



To suggest a resource:

1\. Open an Issue using the Content Request template

2\. Or directly add a `.md` file to this folder and submit a Pull Request



\*All resources must be free, accurate, and beginner-accessible.\*
