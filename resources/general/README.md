\# General Academic Resources



This folder covers skills that apply across all disciplines the kind of

foundational academic competencies that most courses assume you already have

but rarely teach directly.



\---



\## What's in This Folder



| File | Topic | Level |

|------|-------|-------|

| \[academic-writing.md](academic-writing.md) | Essay structure, citations, thesis statements | Beginner |

| \[study-techniques.md](study-techniques.md) | Active recall, spaced repetition, Pomodoro | Beginner |



\---



\## Recommended Starting Order



Start with whichever is most urgent for you right now. Both files are

self-contained and beginner-accessible.



If you have an exam coming up → \*\*study-techniques.md\*\*

If you have an essay or report due → \*\*academic-writing.md\*\*
