\# Computer Science Resources



This folder contains structured learning resources for core computer science

topics. Files are written with university students in mind particularly

those preparing for exams or filling gaps left by inconsistent course coverage.



\---



\## What's in This Folder



| File | Topic | Level |

|------|-------|-------|

| \[data-structures.md](data-structures.md) | Arrays, linked lists, trees, graphs | Beginner to Intermediate |

| \[operating-systems.md](operating-systems.md) | Processes, memory, scheduling, file systems | Intermediate |

| \[networking-fundamentals.md](networking-fundamentals.md) | IP, DNS, TCP/UDP, HTTP, ports | Beginner to Intermediate |



\---



\## Recommended Starting Order



If you're new to computer science or returning after a gap:



1\. \*\*data-structures.md\*\* — foundational. Everything else builds on this.

2\. \*\*networking-fundamentals.md\*\* — practical and immediately applicable.

3\. \*\*operating-systems.md\*\* — more conceptual, best approached after the above.
